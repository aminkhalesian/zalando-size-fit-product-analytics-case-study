# Sample Data Note

This repository does not use real Zalando customer or transaction data.

The case study is based on a hypothetical data model created for portfolio and learning purposes. The tables represent the type of anonymized e-commerce data a Product Analytics team might use to evaluate a Size & Fit feature.

The assumed data includes:

- Customer-level information
- Order-level information
- Item-level purchase and return data
- Fit Quiz interaction events

No personal customer data, body measurements, or private Zalando business data is included in this project.
